Campus Library Network

Find books, compare libraries, and reserve study space from one portal.

FindMyBook connects students, library managers, and admins to the same C++ backend. The web layer handles the map and interface, while the backend handles search, inventory, bookings, and the shared campus library data.

Student Flow Search books across libraries, locate the nearest library, rent books, and reserve seats.
Library Flow Manage inventory and monitor bookings for one library from the same backend data.
Admin Flow Add libraries, review activity, and oversee the full campus-wide system.
